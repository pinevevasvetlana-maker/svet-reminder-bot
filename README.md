# svet-reminder-bot

Телеграм-бот для напоминаний с выбором контакта.

## Запуск

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Установите переменную окружения с токеном бота:

```bash
export TELEGRAM_BOT_TOKEN=your-token
```

3. Запустите:

```bash
python bot.py
```

## Команды

- `/start` — регистрирует пользователя как контакт.
- `/setname <имя>` — задаёт отображаемое имя контакта.
- `/contacts` — список доступных контактов.
- `/remind <имя> <YYYY-MM-DD> <HH:MM> <текст>` — создаёт напоминание (время указывается в UTC).
- `/help` — справка.

## Пример

```
/remind Иван 2024-12-31 09:00 Позвонить
```

> Важно: бот может писать пользователю только если он сам сделал `/start`.
