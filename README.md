# svet-reminder-bot

Телеграм-бот для напоминаний с режимом пользователя и администратора.

- Пользователи создают напоминания только для себя (`/remindme`, `/repeatme`).
- Только администратор (переменная `ADMIN_CHAT_ID`) может создавать напоминания другим.

## Запуск

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Установите переменные окружения:

```bash
export TELEGRAM_BOT_TOKEN=your-token
export ADMIN_CHAT_ID=123456789
```

3. Запустите:

```bash
python bot.py
```

## Команды

### Для всех пользователей

- `/remindme <YYYY-MM-DD> <HH:MM> <текст>` — разовое напоминание себе (UTC).
- `/repeatme <YYYY-MM-DD> <HH:MM> <интервал_мин> <текст>` — повторяющееся напоминание себе.
- `/my` — список своих напоминаний.
- `/cancelme <id>` — отменить своё напоминание.

### Для администратора

- `/contacts` — список контактов.
- `/setname <chat_id> <имя>` — задать имя контакта.
- `/remind <имя> <YYYY-MM-DD> <HH:MM> <текст>` — разовое напоминание для контакта.
- `/repeat <имя> <YYYY-MM-DD> <HH:MM> <интервал_мин> <текст>` — повторяющееся напоминание для контакта.
- `/list` — список всех напоминаний.
- `/cancel <id>` — отменить напоминание.

## Примеры

```text
/remindme 2024-12-31 09:00 Купить подарок
/repeatme 2024-12-01 10:00 1440 Ежедневный отчёт
/my
/cancelme 3
```

```text
/contacts
/setname 123456789 Иван
/remind Иван 2024-12-31 09:00 Позвонить
/repeat Иван 2024-12-01 10:00 60 Напоминание каждый час
/list
/cancel 7
```

> Важно: бот может писать пользователю только если он сам сделал `/start`.
